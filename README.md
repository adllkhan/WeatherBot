## Instalation

> Docker

```bash 
git clone https://github.com/adllkhan/WeatherBot.git | cd weather_bot
```
```bash
docker build -t weather_bot .
```
```bash
docker run weather_bot
```
---
> Python
```bash 
git clone https://github.com/adllkhan/WeatherBot.git | cd weather_bot
```
```bash
python -m venv venv
source venv/bin/activate  # winda venv\Scripts\activate
```
```bash
pip install -r requirements.txt
```
